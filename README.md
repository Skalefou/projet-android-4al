## Installation

Avant toute chose:

1. Installe les dépendances:
   ```bash
   npm i[.gitignore](.gitignore)
   ```

2. Modifie le fichier `.env` selon ta configuration

3. Lance le serveur mock avec:
   ```bash
   npx json-server --watch db.json --port 8080
   ```

Le serveur sera accessible sur le port 8080
